# 🚀 Módulo 3 - Diplomatura en Control de Calidad de Software (QA)

Bienvenido a mi repositorio de prácticas del **Módulo 3** correspondiente a la [Diplomatura en Control de Calidad de Software dictada por la UNTREF](https://untref.edu.ar/carrera/diplomatura-en-control-de-calidad-de-software).

Este espacio está destinado a almacenar mis ejercicios, resoluciones y avances utilizando Python. Este módulo representa casi el último escalón de esta formación profesional, consolidando los conocimientos técnicos de cara al cierre de la diplomatura.

## 🎯 Objetivo del Módulo
El enfoque principal de este repositorio es la **Automatización de Pruebas (QA Automation)**. A lo largo del cursado, estaré subiendo los distintos scripts y ejercicios prácticos desarrollados en **Python**, orientados a asegurar la calidad y optimizar los procesos de testing.
